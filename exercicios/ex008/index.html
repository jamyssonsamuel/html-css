Principais formatações

1. Negrito / Destaque

Tag não semântica:

Nessa frase temos um termo em negrito usando a tag B

Tag semântica:

Nessa frase, temos um Termo em negrito usando a tag STRONG


2. Itálico / Ênfase

Tag não semântica:

Nesta frase temos um termo em itálico usando a tag I

Teg Semântica:

Nessa frase temos um termo em ênfase usando a tag EM

3. Texto Marcado

Podemos criar tambem um texto marcado usando a tag mark

E no outro parágrafo temos outro texto marcado no final

4. Texto grando e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

5. Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

6. Texto Inserido

Podemos marcar um texto como inserido, para dar uma ênfase e indicar que ele foi adicionado depois

Podemos usar também a tag U para um texto sublinhado (não semântico).

7. Texto Sobrescrito

Para inserir do tipo: x2+3

8. Texto Subescrito

Para inserir coisas do tipo: H2O
